# GLADE
GLADE est un compilateur python vers c++ pour accéléré grandement la vitesse du python tout en gardant sa syntaxe
## Élément pris en charge

 - include
 - print
 - fonction
 - return
 - boucle while
 - boucle for
 - if

## Élément en dévlopement

 - commantaire *(de **#** à **//**)*
 - type de variable
 - auto main
 - liste
 - classe
 - str *?*
